# VIP视频解析与3D播放平台
基于静态页面 + Netlify Serverless Functions 构建的视频解析平台，支持B站/爱奇艺/腾讯视频等平台VIP视频解析，可直接部署到Netlify。

## 功能特性
1.  视频库管理（添加/编辑/删除/批量操作）
2.  VIP视频自动检测
3.  多线路视频解析
4.  多画质切换
5.  视频播放与下载
6.  响应式设计，支持移动端

## 部署步骤
1.  Fork 本仓库到个人GitHub账号
2.  登录Netlify，关联该GitHub仓库
3.  无需额外配置，Netlify自动完成部署
4.  部署完成后，访问Netlify分配的网址即可使用

## 注意事项
1.  本项目仅用于学习交流，请勿用于商业用途
2.  解析接口为公开接口，稳定性不保证，可自行替换为私有接口
3.  请勿解析侵权视频，遵守相关法律法规
